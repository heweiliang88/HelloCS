安装和使用

安装

npm安装

```bash
npm install smooth.css --save
```

yarn安装

```bash
yarn add smooth.css
```

cdn使用

```css
<head>
  <link
    rel="stylesheet"
    href=""
  />
</head>
```

基本使用

```
<h1 class="">
```
